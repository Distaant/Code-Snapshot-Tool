# Code-Snapshot-Tool
Program to take automatic screenshots of code and its results for my classes.

The webpage uses the js for the backend.

!!ALL CODE INPUTED WILL RUN WITHOUT RESTRICTION IN THE BACKEND, BE CAREFUL!!

Once set up you can select the theme to change the theme.
You can import:
  1. Exercices PDF (Will probably only work with mine).
  2. A Singular Java project (will import its code automatically, probably will only work with mine).
  3. All the exercices (Java Project.zip) and will sync with the names of the exercices.
  4. An old PDF to continue on top of it.

Auto-Run will autorun the code whenever you change exercice page.
Add to report (cntrl + enter) will take a snapshot of the code and move to the next one.

Download will let you:
  1. Download a png of the current code.
  2. Download a pdf of the current code.
  3. Download all the reports in one PDF.

You can change the titles and descriptions if you want, it will work withot importing anything you can just paste your code
